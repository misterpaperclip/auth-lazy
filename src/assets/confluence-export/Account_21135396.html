Account

The "Accounts" section is intended to manage accounts of the system. Accounts are represented as a table. Use the "Columns" sub-menu of the context menu, to adjust the fields displayed in it.

The "View XXXX Details" will give you the personal and configuration data of selected account, it also allows the user to modify account configuration, trading, and other transactional operations.

The "New Account" command will allocate new account.

Account Details

When executing the "View XXXX Details" menu it will prompt the same tab in Registering new account, but it already filled by the data of selected account. It will give the user the ability to modify and update user details.

Transactions and Balance

Transaction and balance is intended to manage trading and balance operations including multiple balance currencies.

Trades

Information about open trades and pending orders, using this tab, user can see summary of transactions, it can also see the detailed of each account’s transaction. User can also modify, update, and close transactions using this particular tab.

Using the “Add Trade” button, the manager can open a trade using market price.

In Action tab, you can “Modify” and “Close” the specific open transactions.

Add trade

Information on opening trade;

“Instant” will open transaction in Market execution, it will open a trade using the Market price. “Limit” is for buy and sell limit, while “Stop” is for buy and sell stops. (Note that it’s required to fill prices for limit and stop for execution).

Modify Ticket

Information on modify pending order.

Security

The "Security" tab makes it possible to check password of the account and to change the password of the account, as well.

  • Password must be rather complex: at least 5 characters long and containing at least two of three character types (lowercase, uppercase, or digits). The maximum password length is 15 characters.

  • Once the password is changed, the connection of the account to the trader server is reset. A reconnection with the new password is required.
